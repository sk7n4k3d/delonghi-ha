# De'Longhi Coffee for Home Assistant

> **This project is currently on standby.** The integration is under active development and not yet ready for general use. Core functionality (power control, beverage brewing, sensors) works but reliability depends on the Ayla IoT cloud MQTT session which can be unstable on older machine firmware. Use at your own risk.

If you're interested in this project, star the repo to be notified when development resumes.
